# NewsDaily
A news app where users can select category then get the news, also search for a news and save articles.

# Features
- News divided into different categories
- Search functionality
- Favorite articles

# Technology and features
- Generic Network Layer
- MVVM-C Architectural Pattern with Delegation
- Core data for saving articles
- Programmatic UI with SnapKit
- Pagination
- Localization(EN/TR)
- Dark mode support

# Third Party Libraries
- Snapkit
- SDWebImage
